1. GENERAL INFO:
Script reads list of short ID's form txt file, translates them to long ID's and writes xlsx file with both short and long ID's
2. INPUT:
- .txt file containing short ID's. No header needed. No quotes or commas needed. Just insert one ID in each row.
You need to set several variables in the script
- path: Path to your folder
- input_file_name: Name of the input file (containing list of ID's)
3. OUTPUT:
  - "short to long.xlsx" file, containing both short and long ID's
